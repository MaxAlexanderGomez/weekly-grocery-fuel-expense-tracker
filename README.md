# weekly-grocery-fuel-expense-tracker
A Python program that tracks weekly food and gas expenses and counts how many days spending exceeds specified limits.


## Overview

This project collects food and gas expenses for each day of the week, determines how many days spending exceeds specified thresholds, and displays the weekly totals.

## Features

- Collects daily food expenses
- Collects daily gas expenses
- Counts days where food spending exceeds $20
- Counts days where gas spending exceeds $10
- Uses conditional statements
- Demonstrates user input and variables

## Python Concepts Used

- Variables
- If statements
- User input
- Integer data types
- Counters
- Console output

## Technologies

- Python 3

## How to Run

```bash
python weekly_expense_tracker.py
```

or on Windows

```bash
py weekly_expense_tracker.py
```
